# GameWired

GameWired trata-se de um projeto facultativo com fim de aplicar as habilidades de desenvolvimento WEB ensinadas no primeiro semestre (2024.2). O projeto foi realizado em conjunto, com 5 pessoas participando ativamente nele. A continuidade do projeto e site acontece no segundo semestre (2025.1), tendo sido implementado até aqui os conteúdos ensinados no primeiro semestre.
